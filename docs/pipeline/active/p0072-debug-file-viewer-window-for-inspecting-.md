## Stub

A separate window available in debug builds that lets the user browse ~/.claude and other directories to inspect what the Claude binary is doing underneath the SDK. Key capabilities: browse file trees (especially ~/.claude/projects/, session archives, etc.), view JSONL log files with pretty-printing or other legibility transforms, access other debugging output from the Claude process. This is a developer-facing tool for understanding the runtime behavior of the Claude binary that ClodKit wraps.
