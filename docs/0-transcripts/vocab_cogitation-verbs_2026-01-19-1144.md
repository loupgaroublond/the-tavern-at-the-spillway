## Usage Notes

- Mix tones: don't make it all comic or all scholarly
- Some verbs work better for certain agent types (Jake = comic, worker = scholarly)
- Parenthetical explanations can be shown on hover or first use
- Consider seasonal/holiday variants (Pesach-cleaning, Purim-tumbling, etc.)
- Some words are obscene or near-obscene (shtupping) — flag for user preference
- **Don't force -ing form**: Where awkward, use natural forms (e.g., "farblondzhet" not "farblondzhet-ing", "farklempt" not "farklempt-ing"). The display can be "Status: farblondzhet" rather than "farblondzhet-ing..."
- **Obscene words OK**: User approved inclusion of vulgar terms
